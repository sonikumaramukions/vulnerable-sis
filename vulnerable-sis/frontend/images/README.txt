TRIPATHI GROUP OF INSTITUTIONS
Logo Upload Instructions
==========================================

IMPORTANT: Upload your logo file here before deploying to Azure.

1. Place your TRIPATHI GROUP OF INSTITUTIONS logo file here
2. Name it exactly: logo.png
3. Recommended size: 200x200px or larger (will be scaled automatically)
4. Supported formats: PNG (recommended), JPG, SVG

The logo will be displayed on:
- Login page (centered at top)
- Dashboard header (left side)
- All pages with header navigation

After uploading:
- Commit the logo.png file to git
- Push to Azure for deployment

Note: If logo.png is not found, the logo will be hidden gracefully
(no broken image icon will appear).
